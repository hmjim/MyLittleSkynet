> **The Elite Autonomous OS Agent.** Powered by Gemini 3.1 Flash & Groq.

**OpenClaw Lite** is a flagship-grade autonomous agent built for 2026 workflows. It thrives on Windows, operating not as a mere chatbot, but as a **Senior Software Architect** in your terminal. It sees, hears, codes, and navigates the web autonomously to deliver high-precision results.

## 🚀 Key Features

- 👁️ **Visual Mastery**: Full OS & Browser screenshot analysis using Gemini 3.1 Vision.
- 🎙️ **Voice Intelligence**: High-speed voice transcription via Groq (Whisper-v3).
- 🐍 **Internal Code Engine**: Dynamic Python Runner for advanced math and data processing.
- 🌐 **Interactive Browsing**: Playwright-powered navigation (Click, Type, Screenshot).
- 💾 **Persistent Memory**: SQLite-backed conversation history.
- 📤 **Telegram Gateway**: Seamless file response support (Screenshot & Report delivery).
- 🧠 **Senior Engineering Mindset**: Operates with SOLID principles and proactive debugging.

## 🛠️ Tech Stack

- **Models**: [Gemini 3.1 Flash Lite](https://openrouter.ai/google/gemini-3.1-flash-lite-preview), [Groq LLaMA 3](https://groq.com/)
- **Core**: TypeScript, Node.js, Telegraf
- **Automation**: Playwright, Screenshot-Desktop
- **Database**: SQLite3

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/openclaw-lite.git
   cd openclaw-lite
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Setup `.env`:
   ```env
   TELEGRAM_BOT_TOKEN=your_token
   OPENROUTER_API_KEY=your_key
   GROQ_API_KEY=your_key
   ```
4. Install Playwright Browsers:
   ```bash
   npx playwright install chromium
   ```
5. Start the engine:
   ```bash
   npm start
   ```
